# basicSocket
- basic socekt server <-> client 

## how to use
```
$ npm install
$ npm start
```

## Dependency
```
- socket.io     -- socket 
- express       -- server
- ejs           -- template
```

## setting 
```
view -> views -> index.html
router -> router -> main.js
public -> css, js 
```

## local start
```
$ npm start
```

## develop start
```
- npm run-script devlop
```
